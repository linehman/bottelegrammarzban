نکات مهم

  1. دیتابیس را باید از سی پنل ساخته و اطلاعات را در این صفحه وارد کنید
  2. توکن ربات نباید از قبل وبهوک شده باشد
  3. پس از زدن دکمه نصب ربات به صفحه عملیات نصب منتقل می شوید که به شما باید سه پیام موفقیت نصب نشان دهد
  4. اضافه کردن پنل مرزبان از داخل ربات انجام می شود
  5. توکن ربات را باید از ربات @BotFather بگیرید
